比例控制 (P-Control)

賦予機器人「直覺」：學習如何利用數學公式讓無人車根據誤差大小，聰明地調整轉向力道。

🧠 1. 什麼是比例控制 (P-Control)？ (20 min)

比例控制 (Proportional Control) 是回饋控制最基礎的形式。核心哲學是「修正力道與偏差成正比」。誤差越劇烈，系統反擊的力道就越猛。

Control Philosophy

  • 線性映射：透過 $Kp$ (Gain) 將物理世界的偏差轉換為馬達的電氣指令。
  • 負回饋循環：不斷測量誤差並施加修正力，直到目標對齊「零誤差」中心。

Mathematical Core

U(t) = Kp × e(t)
Output = Gain × Error

The Proportional Response Law

🎛️ 2. 比例增益 Kp 的行為特徵 (30 min)

$K_p$ 決定了系統的「個性」。增益過高會導致系統過度反應而產生震盪 (Oscillation)；增益過低則反應遲鈍，導致車體在彎道中衝出軌跡。

Stability Profile

SMALL Kp: Slow Convergence
LARGE Kp: Unstable Oscillation

Waveform Simulation

Limit Cycle & Critical Stability

📉 3. 穩態誤差 (Steady-State Error)

P 控制的原生缺陷：當誤差極小時，修正力量也會趨於零，導致其無法克服物理阻力（如地面摩擦）。這使得系統永遠「差一點點」無法對準中心，這就是穩態誤差。

Physical Reality Check

  • 阻力平衡點：當馬達輸出力道等同於摩擦力時，車體便會停止修正，儘管誤差依然存在。
  • 系統延遲 (Lag)：單純的 P 控制難以應對快速移動目標，容易造成追蹤軌跡的整體偏移。
  • 積分預告 (I-Preview)：理解為消除此誤差，未來我們需要能「累積偏差」的 Integral 控制器。

System Gap

Δ≠0

"誤差是推動力，但微小的誤差推不動沉重的現實。"

GitHub Classroom 詳細操作指南

本單元教學重點：讓學生體驗「控制」不是魔法，而是感知（影像）與執行（馬達）之間的比例橋樑。

  1. 重點 1：震盪的代價 - 許多學員會追求極速修正。必須讓他們看到大 Kp 造成的蛇行 (Oscillation) 如何浪費能量並損害機械壽命。
  2. 重點 2：數據視覺化 - 強力推薦學生使用 Chrome 控制台繪製 `console.log` 的曲線，讓看不見的「增益」變成看得見的「軌跡」。
  3. 重點 3：埋下伏筆 - P 控制是 PID 的 1/3。不必追求完美，重點在於發現 P 的缺陷（穩態誤差），為下一單元引出積分 (I) 做好準備。
