⏳ 1. Fetch 的進化

基於 Promise，通訊變得直覺簡潔。

Why Fetch?

  • Promise 驅動：告別回呼地獄。
  • 語義化介面：函式名稱即是動作。
  • 原生支援：無需外部套件。

Promise Lifecycle

Pending

請求發送中（等待）。

Fulfilled

獲取數據。

Rejected

失敗（斷網）。

✨ 2. Async / Await：優雅

非同步變同步順暢。

Step 01

等待 Header

回傳狀態碼。

Step 02

解析 Body

轉為 JSON。

CLEAN CODE 
async function load() {
  // 1. 送出請求
  const res = await fetch(url);
  
  // 2. 解析 JSON
  const data = await res.json();
}

🛡️ 3. 攔截與逾時

優先處理「異常」。

容錯策略

  • res.ok 檢查：攔截 404/500。
  • try/catch：處理斷網失敗。
  • AbortController：逾時保護。

Network Sentry
