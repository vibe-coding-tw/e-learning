🛡️ 1. 非同步：網頁守門員

🐌 同步 (Sync)

必須在 loop() 呼叫。忙碌時延遲。

⚡ 非同步 (Async)

事件驅動。自動處理不佔 loop()。

Async Logic Analogy

美食街領取號碼牌 (Callback)。櫃檯不需死等。

🗺️ 2. 路由與 Lambda 魔法

利用 C++11 的 Lambda (匿名函式)。直接定義處理邏輯。

Request Handling

  • 路徑對應：對準特定功能區。
  • 非阻塞回應：發送後立即釋放。
  • 參數捕獲：提取即時數值。
LAMBDA SYNTAX 
server.on("/on", HTTP_GET, [](AsyncWebServerRequest *req){
  digitalWrite(LED, HIGH);
  req->send(200, "text/plain", "LED is ON");
});
