⚖️ 1. 什麼是影像質心？ (25 min)

影像質心 (Centroid) 是描述物體位置的最核心指標。它將成千上萬的像素簡化為一個單一的座標點。

Mathematical Definition

$$\bar{x} = \frac{\sum_{i=1}^{n} x_i}{n}, \quad \bar{y} = \frac{\sum_{i=1}^{n} y_i}{n}$$
  • 幾何平衡點：質心是物體的平均位置座標。
  • 降維打擊：將 2D 矩陣數據轉化為 1D 點訊息。

🔭 3. ROI、誤差與控制連動

誤差是控制系統的靈魂。將影像中心與質心的偏差轉化為轉向角指令。透過 ROI (感興趣區域) 排除雜訊干擾。

Control Pipeline

  • ROI 裁切：只關注賽道範圍，無視天空或背景。
  • 誤差計算 (Error)：$e = \bar{x} - Center$。
  • 回授控制：將 $e$ 映射到 Servo 轉向 PWM。

任務 1：實作「質心追蹤器」函式 (Build Centroid Tracker)

角色扮演：視覺導航工程師 | 難度：⭐️⭐️ | 估計時長：60 min

🛠️ 執行技術流程 (Git Workflow)

  1. 建立倉庫：從 GitHub Classroom 連結建立個人作業倉庫並 Clone 到本地。
  2. 像素遍歷實作：在 Tracker.js 撰寫一個迴圈遍歷二值化後的 ImageData。當像素值為 0（黑色目標）時，累加 X, Y 座標。
  3. 中心點計算：計算平均座標 $( \bar{x}, \bar{y} )$，並加入 if (count > 0) 邏輯防止除以零崩潰。
  4. 標註整合：調用 Canvas API 在畫面的質心座標繪製紅色準星，驗證視覺解析正確性。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 座標運算精準度：紅色準星是否能準確鎖定在單一色塊的幾何中心？
  • [ ] 程式穩健度：當影像中「完全沒有黑色像素」時，程式是否能優雅保持（不報錯、不報 NAN）？

任務 2：「質心穩定化」過濾器 (Centroid Stabilization Filter)

角色扮演：訊號處理專家 | 難度：⭐️⭐️ | 估計時長：60 min

🛠️ 執行技術流程 (Git Workflow)

  1. 面積門檻實作：設定 MIN_PIXELS = 50。若偵測到的目標像素數低於此值，則維持上一次的質心座標。
  2. 低通濾波演算法：實作 current = (target * 0.7) + (prev * 0.3)，實現加權移動平均。
  3. 效能對比紀錄：在 README 中記錄加入濾波器後，質心在物體靜止時的座標標準差 (Variance) 變化。
  4. Push PR：提交具備抗噪能力的穩定質心算法。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 抗噪有效性：在環境光線不穩引起像素閃動時，準星是否依然能保持穩定不劇烈晃動？
  • [ ] 追蹤靈敏度：進行低通濾波後，準星對物體快速移動的跟隨延遲是否仍在可接受範圍內？

任務 3：質心誤差與轉向控制連動 (Centroid-to-Control Linkage)

角色扮演：機器人系統整合師 | 難度：⭐️⭐️ | 估計時長：45 min

🛠️ 執行技術流程 (Git Workflow)

  1. 誤差映射實作：計算水平偏離值 $e = \bar{x} - 160$。將 $e \in [-160, 160]$ 映射至舵機角度 $A \in [60, 120]$。
  2. 通訊整合：當 $e$ 發生變化時，透過封裝好的 BLE 介面發送 {steer: A} 到 ESP32 韌體。
  3. 端到端聯調：物理手動移動地面標誌組，驗證無人車車輪是否能隨著標誌位置自動左右跟隨。
  4. Push Final：提交具備「視覺感知與物理執行聯動」的最終專案包。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 物理反應正確性：當質心位於畫面左側（Error 為負值）時，舵機轉向角度是否正確。
  • [ ] 系統集成度：是否成功打通了從影像二元化到物理底層控制的完整閉環 (Close-loop)？

GitHub Classroom 詳細操作指南

本單元教學重點：讓學生理解「影像處理」的終極目的不是看照片，而是提取「可用於控制的座標數據」。

  1. 重點 1：從像素到座標 - 不要讓學生迷失在 `for` 迴圈中，要引導他們思考：當我們找到質心時，我們實質上掌握了物體在世界中的「地址」。
  2. 重點 2：穩定性的價值 - 多數學生會困惑「為什麼準星一直在抖」。這是引入數位濾波與雜訊抑制概念的絕佳契機。
  3. 重點 3：誤差就是驅動力 - 建立 Error 與 Control 的簡單映射關係，為下一個單元的 PID 控制理論鋪路。
