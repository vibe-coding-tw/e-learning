基礎 8 - 運動學轉換

差速轉換數學：
Unicycle Model

將速度與轉向指令。精準解構成左右馬達動力輸出來源。

🚲 1. 什麼是 Unicycle Model？

將機器人簡化為一個運動點。我們控制兩個宏觀參數：

v 線速度 (Linear)

代表整體前進速度。搖桿 Y 軸。

ω 角速度 (Angular)

代表轉向速度。搖桿 X 軸。

Velocity Decomposition

建立直覺操控基礎：將「去哪」與「怎麼轉」解耦。

MotorMix = f(v, ω)

⚙️ 2. 差速驅動與混控公式

左右兩輪獨立驅動。轉向必須在輪間產生「速度差」。

\[L = v - \omega\]
Left Motor
\[R = v + \omega\]
Right Motor
MIXER LOGIC (C++) 
void applyMix(float v, float w) {
  float left = v - w;
  float right = v + w;
  motorL.setSpeed(left * 255);
  motorR.setSpeed(right * 255);
}

🛑 3. 飽和處理：限幅思維

若馬達已最大，直接裁切 (Clipping) 導至轉向失靈。

Normalizing Mixers

  • 等比例縮放：溢出時同步縮小兩輪，保留轉向比。
  • 防裁切機制：全速行駛仍能順滑蛇行。
  • 飽和度判定：使用 max(|L|, |R|, 1.0) 作分母。

Robust Control

"比例大於絕對值。在極限保留轉向意圖。"
