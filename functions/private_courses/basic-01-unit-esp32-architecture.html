基礎 1 - 核心架構

ESP32 架構與開發板導覽

深入了解數位大腦：雙核、Wi-Fi 與豐富外設。通往物聯網世界的鑰匙。

⚡ 為什麼選擇 ESP32？

相較於傳統 Micro-controllers，ESP32 是一個質的飛躍。

🧠

雙核心 (Dual-Core)

運算速度高達 240MHz。一核處理通訊，一核專注執行邏輯。

📡

無線連線能力

內建 Wi-Fi 與 雙模藍牙。這是物聯網神器的核心。

🔌

豐富的外設

內建觸控、感應、多組序列接口。輕鬆驅動馬達與讀取感測器。

🔬 開發板外觀大解剖

ESP-WROOM-32
CE FCC
1

ESP-WROOM-32 模組

藏著 CPU 與 Flash，負責保護精密晶片不受電磁干擾。

2

橋接晶片

翻譯官，負責將 USB 訊號轉換為 UART 訊號通訊。

3

LDO 電壓調節器

將 5V 降至 3.3V。過高電壓會直接燒毀核心。

💻 軟體環境預熱

開始編碼前，必須確保電腦能夠識別連上的裝置。

1. 安裝專業 IDE

推薦 VS Code + PlatformIO。提供自動補全與強大專案管理。

2. 識別通訊埠

安裝正確驅動。沒有它，電腦就只是在對著磚頭講話。

新手防狼手冊

  • 拒絕純充電線： 必須具備「數據傳輸」能力。

  • 物理保護： 捏住接頭插拔，避免拉斷 USB 焊點。

💡 Tip: 如果不亮燈，先檢查電源指示 LED。

核心挑戰：對話的第一步

驗證開發環境，並與晶片建立連結。

💡 45 min

【作業 1-1】

「摩斯密碼閃爍」

安裝環境並燒錄 Blink。修改 logic 讓 LED 閃出「SOS」訊號。

🕵️ 60 min

【作業 1-2】

「晶片身分證偵查」

撰寫 API 查詢代碼，提取 Cores 與 MAC Address，建立資源邊界認知。

🌡️ 45 min

【作業 1-3】

「啟動日誌診斷」

分析一秒 Bootloader 日誌。監測物理溫升，理解軟硬體關聯。
