Developer Notes

UUID Configuration

  • Service: 6E400001-B5A3-F393-E0A9-E50E24DCCA9E
  • Char (SSID): 6E400002... (Write)
  • Char (Pwd): 6E400003... (Write)

Key Logic

TextEncoder is used to convert strings to Uint8Array before writing to characteristics.

Device auto-restarts upon receiving the password (implied commit).

B. 參考範例代碼 (Reference Example Code)

main.cpp (BLE 接收並寫入 NVS)

// 1. 定義 BLE UART Service
#define SERVICE_UUID           "6E400001-B5A3-F393-E0A9-E50E24DCCA9E"
#define CHARACTERISTIC_UUID_RX "6E400002-B5A3-F393-E0A9-E50E24DCCA9E"

// 2. 回呼函數處理接收到的資料
class MyCallbacks: public BLECharacteristicCallbacks {
    void onWrite(BLECharacteristic *pCharacteristic) {
      std::string value = pCharacteristic->getValue();
      if (value.length() > 0) {
        // 解析指令範例: SSID:MyWiFi
        if (value.find("SSID:") == 0) {
            String ssid = String(value.c_str()).substring(5);
            saveToNVS("wifi_ssid", ssid.c_str());
        }
        Serial.println("Received Record: " + String(value.c_str()));
      }
    }
};
