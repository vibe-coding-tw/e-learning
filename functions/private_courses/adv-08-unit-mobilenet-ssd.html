🧬 1. MobileNet：神經網路的瘦身專家 (30 min)

MobileNet 透過「深度可分離卷積 (Depthwise Separable Convolution)」將運算量縮減至傳統卷積的 1/8。它是嵌入式 AI（如 ESP32-S3）實現即時分類的核心架構。

Structural Efficiency

  • 通道拆解：將一次性的 3D 卷積拆分為 2D 空間過濾與 1D 通道融合。
  • 精準度權衡：透過 Width Multiplier 調整模型寬度，平衡記憶體佔用與辨識質量。

⚖️ 3. 權衡之美：FPS 與 mAP

在自動導航中，FPS (反應速度) 往往比 mAP (精準度) 更重要。毫秒級的延遲可能導致避障失效。我們必須學會如何調配硬體加速與模型複雜度。

Engineering Metrics

  • 信心門檻 (Confidence)：過高導致漏失目標，過低引發誤判抖動。
  • WebGL 加速：利用 GPU 並行計算大幅提升推演 (Inference) 效率。
  • 資料清洗：處理 [x, y, w, h] 數據流，實現從對象到距離的空間理解。

AI 實驗室：輕量化模型部屬與數據驅動決策

任務 1：MobileNet 效能診斷 (Inference Benchmarking Lab)

角色扮演：AI 優化工程師 | 難度：⭐️⭐️ | 估計時長：60 min

🛠️ 執行技術流程 (Git Workflow)

  1. 建立倉庫：從 GitHub Classroom 連結建立個人作業倉庫並 Clone 到本地。
  2. 環境預熱 (Warming-up)：載入 TensorFlow.js，並先進行一次空推論以喚醒 GPU。
  3. 信心閾值分析：測試至少 5 種日常物品。記錄信心分數。在剖析報告中說明「背景雜訊」如何影響 Top-1 分類正確率。
  4. 提交報告：在 README 中彙整 WebGL vs CPU 在同一個物體辨識下的延遲 (Latency) 數據對比。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 效能理解度：學員是否能正確辨識出硬體加速帶來的效能提升量級？
  • [ ] 判斷嚴謹性：針對誤判 (False Positive) 案例是否提出了合理的視覺環境解釋？

任務 2：SSD 數據過濾器 (Detection ROI Filter)

角色扮演：視覺邏輯開發者 | 難度：⭐️⭐️⭐️ | 估計時長：60 min

🛠️ 執行技術流程 (Git Workflow)

  1. 數據清洗：實作一個過濾函式，剔除信心門檻低於 0.6 的雜訊框。
  2. ROI (Region of Interest)：在 Canvas 中心繪製一個矩形警戒區。
  3. 幾何碰撞檢測：計算 SSD 回傳的 Bounding Box 中心點是否位於 ROI 矩形內部。
  4. Push Final：提交具備「目標鎖定」與「入域警報」功能的偵測框架。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 數據處理力：學員是否展現了處理 Array 數據流（多個偵測框）的高效邏輯？
  • [ ] 座標轉換精準度：Bounding Box 的幾何計算是否正確地對應了畫面的像素座標？

任務 3：導航行為樹預演 (Behavior Decision Logic)

角色扮演：系統決策工程師 | 難度：⭐️⭐️⭐️ | 估計時長：45 min

🛠️ 執行技術流程 (Git Workflow)

  1. 狀態機設計：根據偵測標籤（Person, Obstacle, Empty）定義無人車的三種狀態。
  2. 誤差映射：將 box.x_center 映射為轉向修正 $X_e$。將 box.width 映射為距離預估 $d$。
  3. 應急停擋實作：當偵測到「Person」且 $d$ 小於閾值時，強制馬達輸出 0（停止）。
  4. Push Final：提交具備安全優先權的 AI 避障邏輯。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 決策穩定性：當模型快速閃爍 (Flicker) 時，控制邏輯是否有防止車體頻繁抖動的平滑技術？
  • [ ] 優先級賦予：是否正確實作了「行人絕對優先」的避讓規則？

GitHub Classroom 詳細操作指南

本單元教學重點：讓學生跨越「影像處理」進入「人工智慧推論」。強調模型不是魔法，而是機率與統計的疊加結果。

  1. 重點 1：速度是關鍵 - 無論模型多準，如果推論需要 2 秒，在移動中的車端就是不可用的。必須強調 30 FPS 的實戰意義。
  2. 重點 2：數據的質感 - 教導學員如何解析 [x, y, w, h] 陣列。物件偵測回傳的是空間資訊，而非單純的類別。
  3. 重點 3：決策邊界 - 信心分數 0.5 跟 0.9 的差別在哪？引導學員建立自己的過濾原則。
