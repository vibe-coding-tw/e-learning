基礎 8 - 控制數學

曲線優化：
線性 vs. 指數響應

掌握演算法彌補肌肉不足。打造完美手感「人機合一」。

📏 1. 線性響應 (Linear Response)

定義：輸出與輸入成正比（\(y = x\)）。搖桿推多少，出力多少。

👍 優點

直觀、好預測。無反應延遲。

👎 缺點

中心點過於敏感。精密操作極難調教。

Linear Logic

\[f(x) = x\]

Constant Slope: 1.0

📈 2. 指數響應 (Exponential Response)

利用冪函數。輸出隨輸入非線性增長。模擬人類心理需求。

Soft Center

中心區域斜率平緩。容許大幅度微調。

Punchy End

推至邊緣急劇攀升。保留爆發動力。

PRO MIXER (JS) 
// (1-k)x + kx^3
function getExpoResponse(x, k) {
  return (1.0 - k) * x + k * Math.pow(x, 3);
}

// k=0: 線性 / k=1: 指數

🔘 3. 調校實務

沒有公式萬能。需具備即時切換控制特性的能力。

Control Architecture

  • 模式熱切換：一鍵切換精準模式 (Expo) 與直覺模式 (Linear)。
  • 視覺連動：UI 配色應有顯著回應，防止誤判。
  • 極性補償：偶次方務必以 Math.sign(x) 保留方向。
