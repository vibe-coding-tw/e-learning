基礎 8 - 控制數學

搖桿數據的本質：
歸一化、映射與死區

將物理像素轉化為精確指令。掌握操縱感的科學底層。

📏 1. 什麼是歸一化 (Normalization)？

不同設備搖桿半徑不同。如果不歸一化，同樣滑動會產生不同速度。

The Core Concept

將物理量轉換為標準比例 0.0 ~ 1.0。代碼與硬體解析度完美解耦。

Normalization Formula

\[V_{norm} = \frac{Offset}{MaxRadius}\]

Range: [-1.0, 1.0]

🗺️ 2. 數值映射 (Mapping)

歸一化值需轉換為硬體理解區間。「職人級」操縱感誕生關鍵。

PWM Control

0 ~ 1.0 映射為 0 ~ 255 馬達動力。

Servo Control

-1.0 ~ 1.0 映射為 0 ~ 180 舵機。

float lerp(float x, float inMax, float outMax) {
  return (x / inMax) * outMax;

🌑 3. 死區 (Deadzone)

搖桿中央不一定是 0。如果不設死區，馬達會微弱共振抖動。

Noise Suppression Loop

  1. 採樣數值：獲取歸一化座標。
  2. 閥值判定：檢查是否大於 10% 範圍。
  3. 數值切掉：小於閥值強制為 0。
  4. 縮放插值：確保起點平滑。

Safe System

"好的系統，靜止時應像冰塊安靜。"
