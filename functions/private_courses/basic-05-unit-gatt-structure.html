🏛️ 1. 藍牙的階層美學：GATT 結構

GATT 定義數據交換。像個嚴密的圖書館：

📦 Profile

角色完整定義。如：「無人車」。

📂 Service

功能邏輯分類。如：「動力」、「傳感」。

📄 Char

數據節點。如：「轉速」、「電量」。

🧠 Coding Tip

將感測與指令分開放在不同 Service，簡化 App 邏輯。

🆔 2. 身份識別：UUID 規格

為了確界不衝突，必須擁有專屬 128-bit 生存碼。

16-bit Standard

藍牙聯盟預定義功能。如心跳 (0x180D)。

128-bit Custom

開發者生成。如自製無人車服務。

🔄 3. 數據交換流派

📥

Read

手機發問。適合穩定數值。

📤

Write

手機下令。用於馬達控制。

🔔

Notify

設備推播。即時遙測方式。

✅

Indicate

附確認通知。安全但慢。
