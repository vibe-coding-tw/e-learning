🛠️ 直觀體驗：從零開始的基建

打地基的時間到了！讓我們用 15 分鐘完成最強編輯器的部署。

1
下載與安裝 (15 min)

前往 code.visualstudio.com，根據你的作業系統 (Windows/macOS/Linux) 下載。

Windows 用戶關鍵設定：

安裝時務必勾選 「加入 '開啟以程式碼編輯' 動作到 Windows 檔案總管內容功能表」 (Add "Open with Code")。

💡 這讓你能在資料夾直接按右鍵開啟專案，效率提升 10 倍！

2
繁體中文介面設定 (5 min)

  • 開啟 VS Code。
  • 點擊左側工具欄最下方的四方塊圖示 (Extensions) 或按 Ctrl+Shift+X。
  • 搜尋 Chinese，選擇 "Chinese (Traditional) Language Pack" 安裝。
  • 重啟 VS Code，介面即刻中文化。

VS Code 安裝與基礎配置

任務 1：安裝硬體大腦 PlatformIO

角色扮演：韌體工程師 | 難度：⭐️⭐️ | 估計時長：30 min

🛠️ 執行技術流程 (Git Workflow)

  1. 搜尋套件：在 VS Code 擴充功能商店搜尋並安裝 "PlatformIO IDE"。
  2. 下載組件：耐心等待 Python 與 PIO Core 元件自動下載與安裝。
  3. 確認狀態：檢查左側工具欄是否出現外星頭 👽 圖示。
  4. 環境就緒：進入 PIO Home 畫面並截圖，確保開發環境已完全就位。

    5. 任務 1：繁體中文化與同步配置 (L10N & Sync Setup)

    角色扮演：環境專家 | 難度：⭐️ | 估計時長：30 min

    🛠️ 執行技術流程 (Git Workflow)

    1. 安裝環境：下載並安裝 VS Code 本地開發環境。
    2. 語系挑戰：於擴充商店搜尋並安裝 "Chinese (Traditional)" 繁體中文語言包。
    3. 同步設定：登入個人 GitHub 帳號並開啟「設定同步 (Settings Sync)」。
    4. 環境驗證：截圖並更新至 README，標註 VS Code 已啟動同步狀態。

    🎯 教師點評標準 (Tutor Benchmarks)

    • [ ] 介面正確性：VS Code 介面是否已正確顯示為繁體中文？
    • [ ] 同步狀態：左下角人頭圖示是否顯示已登入且同步中？

任務 2：PlatformIO 韌體組件安裝 (PIO Extension Lab)

角色扮演：韌體工程師 | 難度：⭐️⭐️ | 估計時長：45 min

🛠️ 執行技術流程 (Git Workflow)

  1. 外掛安裝：在 VS Code 安裝 PlatformIO IDE 官方外掛。
  2. 核心加載：觀察右下角狀態列，等待 PIO Core 加載完成並出現「👽」外星人圖示。
  3. 模擬開啟：開啟 platformio.ini，確認 PIO 是否能正確識別專案環境設定。
  4. Push Final：提交包含開發環境檢測清單的 README 報告。

🎯 教師點評標準 (Tutor Benchmarks)

  • [ ] 外掛運作：左側功能列是否已出現 PlatformIO 的專屬圖示？
  • [ ] 核心識別：platformio.ini 的語法突顯 (Syntax Highlight) 是否正常運作？

GitHub Classroom 詳細操作指南

本單元教學重點：打造堅實的本地開發堡壘。引導學生完成 VS Code 與 PlatformIO 的深度整合，確保後續課程的韌體開發無縫接軌。

  1. 重點 1：母語優先的學習路徑 - 透過繁體中文化，降低初學者對 IDE 英文介面的恐懼感。
  2. 重點 2：工作環境隨身帶 - 強調設定同步 (Settings Sync) 的重要性，讓學生不論在教室還是家中都能擁有一致的開發體驗。
  3. 重點 3：專屬韌體工具箱 - 介紹 PlatformIO 作為專業韌體開發工具，協助學生跨越 Arduino IDE 的功能限制。

教學實施策略 (Flipped Workflow)

階段 學生行動 (Git Workflow) 教師/平台角色
課前預習 根據講義下載 VS Code，完成繁體中文化。 支援：在討論區排除 Windows/Mac 下載版本不相符的問題。
課中實作 課堂安裝 PlatformIO（大檔案下載），並開啟 Sync 同步。 巡堂：檢查誰卡在「PIO Core 載入中」，協助手動除錯。
課後精進 建立一個空白 PIO 專案，上傳至 GitHub 觀察環境是否一致。 Feedback PR：檢核學生提供的 PIO 設定截圖。
