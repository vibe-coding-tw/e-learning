💻 3. switch...case 與 enum

擺脫魔術數字。使用 enum 定義語義。

Clean Architecture

  • 消滅 Magic Numbers：用 IDLE 代替 0。
  • 高擴充性：新增模式只需在 switch 增加 case。
  • 事件驅動：由條件觸發轉移。
enum State { IDLE, AUTO };
State s = IDLE;

switch(s) {
  case IDLE:
    if(btn) s = AUTO;
    break;
  case AUTO:
    motorOn();
    break;
}
