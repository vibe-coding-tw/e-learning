創客的生存指南

認識數位探針：學會區分 GPIO、ADC、DAC。避開會讓系統崩潰的「地雷引腳」。

🗺️ 1. 認識引腳地圖 (Pinout Map)

ESP32 是「瑞士軍刀」。引腳在不同配置下，能變幻出截然不同的身分。

GPIO (General Purpose)

數位開關。讀取按鈕 or 點亮 LED。

ADC & DAC

類比魔法門。處理電壓變化。

Capacitive Touch

內建電容感應。隔著塑膠殼也能操作。

Hardware Bus

通訊高速公路：UART、I2C 與 SPI。

Quick Navigation

🖨️ 需要對應特定機型。

💣 2. 絕對不能忽視的「地雷引腳」

🚫

僅限輸入 (Input Only)

GPIO 34, 35, 36, 39 無輸出。適合感測器，不能推動 LED。

⚙️

啟動配置 (Strapping)

GPIO 0, 2, 5, 12, 15 掌握開機「生殺大權」。接錯電阻，板子就會死鎖。

⚠️

硬體生存守則 (Case Study)

「為什麼專案接上外部電源就燒錄不進去？

通常是因為在 GPIO 0 接了上拉按鈕，導致開機誤入 Boot 模式。」

* 專業建議：優先使用 GPIO 12 以後的通用引腳。

⚡ 3. 供電腳位與安全電流

ESP32 不是電源供應器。過度汲取電流會直接燒毀內部的 I/O 閘道。

Vin

Voltage Input (5V)

外部電源入口，通常由 USB 提供。

3V3

LDO Output (3.3V)

穩定輸出，專供精密感測器使用。
