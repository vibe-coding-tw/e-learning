課程使用說明

這裡整理了課程使用方式與相關服務說明

🚀 完整課程使用指南

1

註冊 / 登入會員

首先，請點擊網站右上角的 「登入」 按鈕。我們支援 Google 快速登入，或使用 Email 註冊帳號。擁有帳號後，系統才能記錄您的購買與學習進度。

✨ 新手好禮：首次註冊後，您可以免費體驗一個月的入門課程！

2

瀏覽並挑選課程

前往 入門課程 或 基礎實作 頁面，瀏覽詳細的課程大綱與介紹。確認符合您的需求後，準備進行購買。

3

加入購物車

在課程頁面點擊 🛒 加入購物車 按鈕。您可以一次挑選多堂課程，最後再一次結帳。

4

確認訂單與付款

進入購物車頁面確認金額無誤，點擊「前往付款」。系統將跳轉至 綠界科技 ECPay 安全支付頁面，您可選擇信用卡或 ATM 轉帳。

5

立刻開始學習

付款成功後，系統將自動導回網站並開通權限。回到課程頁面，按鈕將變更為 「進入課程內容」，點擊即可進入教室開始上課！

如有任何操作疑問，歡迎參考下方常見問題或直接聯繫客服信箱。

🎓 1. 完成購買後，該如何上課?

上課流程非常簡單：

  1. 請點擊網站右上角的「登入」按鈕。
  2. 登入您的帳號（建議使用 Google 快速登入）。
  3. 前往課程頁面，原本的「加入購物車」按鈕會轉變為「進入課程內容」。
  4. 點擊該按鈕，即可開始觀看課程影片與實作教材。

🛠️ 2. 沒辦法順利看課程怎麼辦？

如果您已經購買課程，但遇到按鈕狀態未更新或權限問題，請別擔心。

請直接來信與我們聯繫，我們將優先為您處理：

📧 info@vibe-coding.tw

⏳ 3. 課程的使用期限與服務？

完成課程付款後，您將擁有一年的課程使用權限。

系統會為您分配一位指導老師，協助您解答學習過程中的疑問。

提醒 課程到期前一個月，系統會自動發送 Email 提醒。

💻 4. 課程前需要準備什麼設備？

為了確保最佳學習體驗，我們整理了一份詳細的電腦軟體與硬體清單。

請在開始上課前詳細閱讀：

👉 查看「課前準備事項」

👩‍🏫 5. 老師會提供什麼協助？

老師的主要職責包含：

  • 線上問題解答與技術指導。
  • 批改作業並提供具體的優化反饋。
  • 協助您在 GitHub Classroom 上的實作挑戰。

📦 6. 如何購買硬體教材與運送方式？

您可以在 課前準備 頁面購買 ESP32 套件或 Car Platform 實作車。

  • 超商取貨：結帳時可選取 7-11、全家等門市。
  • 付款完成即寄出：我們通常在確認訂單後的 2 個工作天內出貨。

📊 7. 如何追蹤我的學習進度？

登入後，在課程中點擊「課程儀表板 (Dashboard)」即可查看：

  • 學習時數統計：掌握各單元的累積學習時間。
  • 作業批改狀態：查看老師的分數評語與退回調整建議。

📝 8. 「實作任務」與 GitHub Classroom 如何搭配？

針對程式實作，我們簡化了作業流程：

  • 點擊開始：單元內的按鈕會引導您至 GitHub Classroom 建立您的個人專案。
  • 提交回傳：完成程式開發後，回到平台儀表板按下「提交」，通知老師進行專業批閱。
