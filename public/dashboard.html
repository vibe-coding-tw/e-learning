課程設定 (Settings)

⚡ 如何設定與使用「自建原生 API (Native API)」作業指派模式？

1. 準備 GitHub 存取權杖 (Token)： 請至您的 GitHub 帳號產生一個 Fine-grained personal access token。必須賦予該 Token 對應組織/倉庫的 Administration (Read & Write)、Contents (Read & Write) 以及 Metadata (Read-only) 權限。

2. 於下方單元配置參數： 在下方要指派的單元中，將作業指派方式切換為「自建原生 API」，填入您的 組織名稱 (Org)、教材樣板倉庫 (Template Repo)，並貼上產生的 Token 後儲存。

3. 學生端一鍵自動建立： 請提供您的 Promotion code 給學員綁定。學生點擊「前往教室寫作業」並確認 GitHub 帳號後，系統即會在 3-8 秒內背景自動建立作業 Repository 並將學生加為協作者，免除一切組織加入卡點，徹底解決 Repository Access Issue！
